# 08 - Who's Playing App PRD v2.0

## Product Vision
Create the easiest way to discover local live music while supporting artists and venues.

## MVP Goals
1. Event discovery
2. Artist profiles
3. Venue profiles
4. Search
5. Map-based discovery
6. Event submission/review

## MVP Non-Negotiable
The app must be clean and easy to navigate. It must not look like a design-spec collage.

## User Types
- Fans
- Artists
- Venues
- Admins

## MVP Screens
- Home
- This Week
- Event Detail
- Artist Profile
- Venue Profile
- Map
- Add Event
- Profile

## Design Requirements
- Mobile-first
- Compact logo in header
- Clear event cards
- Fast filtering
- High readability
- No visual clutter
