# 03 - Product Architecture v2.0

## Core Product
Who's Playing is a live local music discovery platform.

## Core Objects
- Events
- Artists
- Venues
- Communities / Cities
- Users
- Saved favorites
- Event submissions

## Core Screens
- Home
- Events
- Event Detail
- Artists
- Artist Profile
- Venues
- Venue Profile
- Map
- Add Event
- Profile

## Important Design Boundary
Product screens must not include brand-review-board artifacts.

Do not render:
- PRIMARY LOGO
- COLOR PALETTE
- TYPOGRAPHY
- Logo construction panels

Those belong only in design documentation or internal review pages.
