# 01 - Who's Playing Brand System v2.0

## Brand Mission
Help music lovers discover live local music while helping artists and venues grow their audiences.

## Brand Promise
Find live music faster. Support local music deeper.

## Approved Brand Direction
Clean, modern, app-ready, and community-focused.

Visual blend:
- Apple-level clarity
- Spotify-like music energy
- Airbnb-like local friendliness

## Approved Logo
The approved primary logo is the **Live Pin Lockup**.

Do not create new logo alternatives.

## Live Pin Meaning
- Location Pin = nearby local discovery
- Play Button = music and action
- Signal Waves = live activity happening now

## Logo Variants
Use these exact variants:

### Full Lockup
Used for:
- Brand documents
- Landing page hero if large logo is intentionally needed
- Marketing materials

Includes:
- Mark
- Stacked wordmark
- Tagline

### Compact Header Logo
Used for:
- Website header
- App shell header
- Navigation bars

Includes:
- Small mark
- Wordmark only

Does not include:
- Tagline
- PRIMARY LOGO label
- Color palette
- Typography panels

### Mark Only
Used for:
- App icon
- Favicon
- Map pins
- Social avatar

## Primary Logo Gradient
Use on the pin and play triangle:

```css
linear-gradient(135deg, #FF4F63 0%, #FF6B42 48%, #FF2F70 100%)
```

## Wordmark Gradient
Use on `playing`:

```css
linear-gradient(180deg, #FF8751 0%, #FF5A5F 52%, #FF3F73 100%)
```
